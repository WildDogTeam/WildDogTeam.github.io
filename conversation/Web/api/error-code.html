错误码 编辑文档

错误码 错误信息 描述
41001 TOKEN_EXPIRED 野狗身份认证 的 token 过期
41002 TOKEN_INVALID 野狗身份认证 的 token 无效
41003 GET_CONFIG_ERROR 从服务器端获取 Client Config 时发生错误
41004 CONVERSATION_CONNECT_FAILED 一对一视频通话建联失败
41005 DEVICE_NOT_AVAILABLE 音视频设备获取失败
41006 OPERATION_TIMEOUT 操作超时
41007 CONVERSATION_TIMEOUT 被叫方由于超时未接听邀请或视频通话成功后由于网络或其他原因发生的错误
