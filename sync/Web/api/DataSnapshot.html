DataSnapshot 编辑文档

wilddog.sync.DataSnapshot 是当前指定节点下的数据快照，DataSnapshot 不会随当前节点数据的变化而发生改变。我们无法直接创建这个对象，而应当在 on 或 once 的回调函数中来获取它。

方法

exists

定义

exists()

说明

判断当前 DataSnapshot 实例中是否包含数据。使用 exists() 方法进行非空判断比 snapshot.val() !== null 更高效。

返回值

Boolean

val

定义

val()

说明

返回当前数据快照包含的数据。val() 可能返回的数据类型包括：String、Number、Boolean、null（该节点下数据为空时）或者是数组、对象。返回的数据类型取决于节点下的数据内容。

返回值

Object|String|Number|Boolean|null 当前数据快照包含的数据， null 表示该节点的数据快照为空。

示例

child

定义

child(path)

说明

根据相对路径，来获取当前节点下子节点的数据快照。相对路径可以是一个字节点的 key 值（如："Beijing"），也可以是更深层次的路径（如：'Beijing/pm25'）。如果相对路径下并没有数据，则返回 null 。

参数
参数名 说明
path String(non-null)类型
path 为相对路径，多层级间需要使用"/"分隔，例如"a/b"。

forEach

定义

forEach(callback)

说明

遍历数据快照中的每一个子节点。受 JavaScript Object 对象的影响，snapshot 直接通过 val() 方法返回的数据不能保证显示的顺序完全符合 orderBy*() 的要求，因此我们提供了 forEach() 方法来解决这个问题。如果没有使用 orderBy*() 方法，依次遍历出来的结果默认选择 key 排序（除非数据快照设置过 priority ，则会根据 priority 排序）。

参数
参数名 说明
callback function(non-null)类型
遍历每一个子节时的回调函数。如果在 callback 中主动 return true 则会停止之后的遍历。
返回值

Void

hasChild(key)

说明

参数
参数名 说明
key String(non-null)类型
要检查的 key 。
返回值

Boolean

说明

返回值

Boolean

示例

假设我们已经有如下的数据：

getPriority

定义

getPriority()

说明

获取当前节点的 priority 值。如果优先级不存在时返回 null。

返回值

String|Number|null 不存在优先级时返回 null。

示例

exportVal

定义

exportVal()

说明

将 DataSnapshot 中的全部内容导出到 JavaScript 对象。exportVal() 方法和 val() 方法类似，都可以导出数据。但是当节点的 priority 值不为空时，exportVal() 会导出包含 priority 的数据，适合用于备份。

返回值

数据快照的值和优先级。
